# 2Team-Pain-Catchers

### 프로젝트 소개
통증 관리 전문가들을 위한 올인원 고객 관리 전문 SaaS

### 역할 분담
도희정 : 차트<br>
이은지 : 화면구성<br>
신주희 : 사진위에 필기<br>
최상현 : 카메라<br>

### 노션

## 기술스택 및 개발환경
### Flutter
- Version - 3.13.9
- [데이터 모델링] : Freezed
- [DI 및 상태관리] : Riverpod , GetIt

- [네비게이션] : GoRouter
- [로컬저장소 - 오프라인 대응 목적 X] : Hive

- Version - 3.1.5

- Version - 1.84.2
